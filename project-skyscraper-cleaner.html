Vacuum Assisted Wall Climber for Skyscraper Cleaning

A pneumatic-based window cleaning prototype that uses vacuum-assisted wall climbing mechanism for cleaning high-rise building windows.

January 2020 - March 2020
PSG Polytechnic College
Team of 4
Diploma Final Year Project
View on GitHub

Project Overview

This diploma final year project presents an innovative pneumatic-based system designed for cleaning high-rise building windows. The prototype demonstrates "vacuum-assisted wall climbing" technology, scaled down from industrial applications for practical testing and demonstration purposes.

The system addresses the growing need for automated cleaning solutions in urban environments where manual window cleaning of skyscrapers poses significant safety risks and logistical challenges. By utilizing pneumatic suction cups and an intelligent control system, the robot can adhere to vertical glass surfaces while performing cleaning operations.

Key Innovation

The vacuum-assisted adhesion mechanism allows the robot to climb vertical glass surfaces without the need for external support structures, making it ideal for modern glass-facade buildings.

Design & Visualization

The system was designed using SolidWorks and Creo Parametric, with detailed 3D models and technical drawings created for manufacturing.

3D CAD Model - Complete Assembly

How It Works

The wall climbing robot operates through a coordinated sequence of pneumatic actuations and vacuum-based adhesion:

1

Vacuum Adhesion

Pneumatic suction cups create a strong vacuum seal against the glass surface, allowing the robot to securely attach to vertical walls.

2

Sequential Movement

Pneumatic cylinders extend and retract in a coordinated pattern, moving the robot upward while maintaining at least one set of suction cups attached at all times.

3

Cleaning Operation

The integrated brush mechanism cleans the glass surface as the robot moves, removing dirt and debris from high-rise windows.

4

Arduino Control

An Arduino microcontroller coordinates the solenoid valves, controlling the timing of vacuum cups and pneumatic cylinders for smooth operation.

Technical Specifications

Hardware Components

Solenoid Valves - MAX-SV61 model for pneumatic control
Pneumatic Cylinders - For linear actuation and movement
Vacuum Cups - Industrial-grade suction cups for adhesion
Mini Vacuum Pump - Creates negative pressure for suction
L-Angle Frame - Modular structural framework
Bush Components - For smooth mechanical movement
Brush Assembly - Cleaning mechanism attachment
Arduino Board - Microcontroller for automation
