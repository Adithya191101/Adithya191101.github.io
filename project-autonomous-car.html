Development of Scalable Autonomous Car

A simulation-based autonomous vehicle project using CARLA simulator, implementing SLAM algorithms to achieve Level 2 autonomy with Indian traffic conditions.

July 2022 - November 2022
PSG College of Technology
Team Project
Autonomous Vehicle Simulation
View Project Report

Project Overview

This project focuses on developing a scalable autonomous car simulation using the CARLA (Car Learning to Act) open-source simulator. Following CARLA's official documentation and baseline implementations, the team built an autonomous driving system capable of navigating through simulated urban environments.

The implementation combines ORB-SLAM2 for visual simultaneous localization and mapping with Deep Q-Network (DQN) reinforcement learning for decision making, achieving Level 2 autonomy where the vehicle can control both steering and acceleration/deceleration.

Project Goal

Achieve Level 2 autonomy in simulation by implementing ORB-SLAM2 for localization and DQN-based reinforcement learning for autonomous navigation and control.

Implementation Approach

The project followed a systematic approach to develop and test the autonomous driving algorithms in simulation.

1

Environment Setup

Configured CARLA simulator to replicate Indian traffic conditions including auto-rickshaws, two-wheelers, pedestrians, and diverse road conditions with varying lane discipline.

2

Sensor Integration

Implemented virtual sensors including cameras, LiDAR, and radar to provide the autonomous system with environmental perception data.

3

ORB-SLAM2 Implementation

Integrated ORB-SLAM2 (Oriented FAST and Rotated BRIEF) for visual simultaneous localization and mapping, enabling real-time camera-based localization and sparse 3D reconstruction of the environment.

4

DQN Agent Development

Implemented Deep Q-Network (DQN) reinforcement learning agent following CARLA's baseline approach, utilizing experience replay and target networks for stable learning of driving policies.

5

Training & Evaluation

Trained the DQN agent through multiple episodes in CARLA environment, optimizing reward functions for lane keeping, collision avoidance, and navigation objectives.
