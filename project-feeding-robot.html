Project Overview

This initiative presents a comprehensive simulation framework for assistive feeding robotics, targeting approximately 6.8 million Americans experiencing upper-limb impairments. The system utilizes a Universal Robots UR5e manipulator to execute secure, seamless spoon-to-mouth feeding trajectories with continuous safety oversight.

Core Technical Innovations

  • Hybrid Motion Planning: Joint-space interpolation for singularity avoidance + Cartesian interpolation for precise feeding motions
  • Sensorless Impedance Control: Force estimation from position tracking errors without expensive F/T sensors
  • MoveIt2 Integration: Semantic IK seeding for consistent, predictable trajectories
  • Phase-Dependent Safety: Adaptive stiffness scaling based on proximity to user

Simulation Environment

The system is built using ROS2 Jazzy with Gazebo Harmonic for realistic physics simulation and MoveIt2 for motion planning.

System Architecture

The control pipeline integrates motion planning, trajectory execution, and real-time safety monitoring in a cohesive architecture.

Cartesian Waypoints → Inverse Kinematics (MoveIt2) → Hybrid Motion Planner → Joint Trajectories → ros2_control → Gazebo Simulation → Joint State Feedback (100 Hz) → Impedance Safety Monitor (10 Hz) → Emergency Stop (if force > 25 N)

Hybrid Motion Planning

Intelligently switches between joint-space interpolation for large reconfigurations (avoiding singularities) and Cartesian linear interpolation for precise feeding motions with orientation preservation.

Singularity-Free <2° Deviation Smooth Trajectories

Sensorless Impedance Control

Estimates contact forces from position tracking errors without expensive force/torque sensors. Implements phase-dependent adaptive stiffness (100 N/m base, 0.15-1.0 scaling) for increased sensitivity near the user's face.

No F/T Sensor Needed 25N Force Threshold Adaptive Stiffness

MoveIt2 with Semantic IK

Phase-aware inverse kinematics seed selection biases the solver toward kinematically suitable solutions, preventing unnecessary joint rotations and ensuring consistent trajectories between waypoints.

KDL Kinematics Collision-Aware Predictable IK

Feeding Sequence

The robot executes a carefully choreographed 8-waypoint sequence optimized for safety and user comfort.

1

Home Position

Robot starts in a safe, neutral configuration away from the user [0.22, 0.0, 0.75]m

2

Safe Height Movement

Moves to elevated position using joint-space interpolation for singularity avoidance

3

Plate Approach

Descends toward plate position with tilted orientation for scooping [0.35, 0.58, 0.52]m

4

Food Scooping

Executes precise scooping motion at the plate with controlled orientation

5

Mouth Approach

Lifts food and approaches mouth region using Cartesian-space for straight-line path [0.43, 0.0, 0.78]m

6

Food Delivery

Delivers food at mouth position with reduced stiffness (0.15 gain) for maximum safety

7

Safe Retraction

Retracts safely away from the user's face before any large movements

8

Return Home

Returns to home position, completing the cycle in approximately 82 seconds

Key Technical Findings

Hybrid Motion Planning Results

  • Transport (Joint-space): Singularity-free robust motion during large reconfigurations
  • Feeding (Cartesian-space): Straight-line paths with <2° orientation deviation
  • Pure Cartesian Failed: Singularities in folded configurations
  • Pure Joint-space: 15° orientation deviations (unacceptable for feeding)

Safety System Validation

  • Nominal Peak Force: 19.12 N (24% margin below 25 N threshold)
  • Emergency Stop Test: Correctly triggered at 45.91 N during challenge scenarios
  • Zero False Negatives: All unsafe conditions detected
  • Signal Filtering: Rolling mean (window=8) essential for eliminating physics noise
